DESCRIPTION
===========

A generic cookbook to manage simple system resources:
* directories
* symbolic links
* users
* locale of machine
* installation/upgrade of packages
* hosts fiel

REQUIREMENTS
============

None

ATTRIBUTES
==========

All node attributes are set under the `system` name space.
* `locale`
* `users`
* `directories`
* `links`
* `packages`
* `hosts`
